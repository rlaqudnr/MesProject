# 🚀 NexMES Pro System


> NexMES는 React와 ASP.NET Core를 바탕으로, 현장의 수주부터 생산, 재고 관리까지 실시간으로 제어하는 제조 실행 시스템입니다.

## 📊 주요기능
* **1** 차량 생산 시 부품 재고 실시간 자동 차감.
* **2** 수주 등록 시 PDA 단말기 즉시 데이터 전송.
* **3** 재고 부족 시 생산 프로세스 원천 차단 로직.
* **4** 로그인/회원가입/세션 및 비밀번호 확인 기반 회원 탈퇴.
* **5** 실시간공정수율계산/차량통계추적
